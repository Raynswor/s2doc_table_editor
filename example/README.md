# @s2doc/table-editor - Example Application

This directory contains an example application demonstrating how to use the standalone table editor component.

## Running the Example

1. Install dependencies (from the root of the project):
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown in the terminal (usually http://localhost:5173)

## Features Demonstrated

- Loading a sample table
- Editing table cells
- Adding/removing rows and columns
- Merging and splitting cells
- Exporting table data to JSON
- Importing table data from JSON
- Cell selection and multi-select

## Integration Example

See `src/App.vue` for a complete example of integrating the table editor into a Vue 3 + Vuetify application.
